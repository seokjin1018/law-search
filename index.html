검색 방법 안내

쉼표(,)를 사용하여 여러 키워드를 입력할 수 있습니다. 띄어쓰기는 검색 시 무시됩니다.

  • 단일 (기본값): 입력한 키워드 중 첫 번째 키워드만으로 검색합니다.
  • OR: 입력한 키워드 중 하나라도 포함된 모든 결과를 찾습니다.
    예: `계약, 해지` → '계약' 또는 '해지'가 들어간 결과
  • AND: 입력한 모든 키워드가 포함된 결과만 찾습니다.
    예: `계약, 해지` → '계약'과 '해지'가 모두 들어간 결과
  • AND + OR: 첫 번째 키워드는 반드시 포함하고, 나머지 키워드 중 하나 이상이 포함된 결과를 찾습니다.
    예: `계약, 해지, 무효` → '계약'은 필수, 그리고 '해지' 또는 '무효'가 들어간 결과
  • 제외: '제외' 입력창의 키워드가 포함된 결과는 어떤 검색에서도 제외됩니다.
